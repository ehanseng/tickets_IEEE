⚠️ Configuración Pendiente

Los archivos se han desplegado correctamente, pero la aplicación Python necesita ser configurada en el servidor.

📋 Próximos Pasos

Opción 1: Si tienes cPanel

1. Ir a "Setup Python App" en cPanel
2. Crear nueva aplicación Python
3. Python version: 3.9+ (la más reciente disponible)
4. Application root: /home/usuario/ticket
5. Application URL: ticket.ieeetadeo.org
6. Application startup file: main.py
7. Application entry point: app
8. Instalar dependencias desde requirements.txt

Opción 2: Si tienes SSH/Terminal

cd /ruta/al/directorio
curl -LsSf https://astral.sh/uv/install.sh | sh
source $HOME/.cargo/env
uv sync
uv run uvicorn main:app --host 0.0.0.0 --port 8000
